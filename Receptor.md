## Receptor properties
|  Name   |                         Description                          |
| :-----: | :----------------------------------------------------------: |
|  x  | The position of the receptor on the X Plane |
|  y   | The position of the receptor on the Y Plane|
| angle | The angle of the receptor |
| alpha | The transparency of a receptor |
